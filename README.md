# Brain Games

Set of console games based on Node.js

---

####Installation
```npm link```

[![asciicast](https://asciinema.org/a/25lNO5UkpWD6rfLuEhFRWdXk3.svg)](https://asciinema.org/a/25lNO5UkpWD6rfLuEhFRWdXk3)


####Uninstallation
```npm unlink```
